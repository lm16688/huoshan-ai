# NovaClip AI - 智能视频剪辑工具（火山引擎版）

本项目基于 React + TypeScript + Vite 构建，使用火山引擎多模态大模型 `doubao-1-5-vision-pro-32k-250115` 对上传的视频进行语音识别、冗余内容剔除和语义分割，最终生成可导出剪辑后的视频。
